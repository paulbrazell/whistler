# Product Overview

**Whistler** is a band website for an indie rock band from Charlotte, NC. The site serves as the primary digital presence for the band, featuring:

- Album promotion for "Everywhere But Here" EP
- Live show listings via Bandsintown integration
- Social media links and contact information
- Merchandise links to Bandcamp store

The website is designed as a single-page application with a clean, modern aesthetic that reflects the band's indie rock identity. The primary goal is to drive music streaming and show attendance while maintaining a professional online presence.

## Key Features

- Album artwork and streaming links
- Integrated show calendar
- Social media connectivity
- Mobile-responsive design
- Fast loading with modern web technologies
